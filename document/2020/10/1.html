# 4个vsCode 常用插件，提升你的团队开发效率

现在前端主流开发工具有WebStorm，vsCode，Atom，SublimeText，HBuilder等，每个编辑器都有自己的优点。为什么是vsCode？
你在 vsCode 中找到的每个功能都完成一项出色的工作，构建了一些简单的功能集，包括语法高亮、智能补全、集成 git 和编辑器内置调试工具等，将使你开发更高效。
vsCode 最重要的功能是它的侧边栏，你需要的其他任何功能都可以通过安装扩展来满足。
vsCode 提供了一个功能齐全的集成终端，可以让你选择终端，并且运行常用命令。

在一个团队中效率最为重要，前端代码规范，这些插件起了很重要的作用，自定义适合自己的编辑器。

# 1、koroFileHeader

在vscode中用于生成文件头部注释和函数注释的插件，完整规范的代码注视

// 文件头部注释 可以根据自己的喜好进行设置
"fileheader.customMade": {
  "Descripttion":"",
  "version":"",
  "Author":"yanghui",
  "Date":"Do not edit",
  "LastEditors":"yanghui",
  "LastEditTime":"Do not Edit",
},
